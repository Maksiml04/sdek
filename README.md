# SDEK Start RAG Agent

Интеллектуальный чат-бот для программы стажировок **СДЭК.Start**, построенный на архитектуре RAG (Retrieval-Augmented Generation). Бот отвечает на вопросы студентов о программе, условиях участия, дедлайнах и жизни в других странах, используя базу знаний из внутренних документов.

## Возможности

- **Контекстные ответы**: Использует векторный поиск по документам для генерации точных ответов.
- **Память диалога**: Поддерживает контекст беседы в рамках одной сессии.
- **Мультиязычность**: Понимает запросы на русском и английском языках.
- **Фильтрация**: Умеет различать общие вопросы и вопросы по конкретным странам (Германия, Франция и др.).
- **Изоляция данных**: Все данные хранятся локально в векторной базе ChromaDB.

## Требования

- **Docker** и **Docker Compose**
- **Python 3.11+** (для локальной разработки)
- API ключ от LLM провайдера (по умолчанию используется OpenRouter)

## 🛠Установка и запуск

### 1. Подготовка переменных окружения

Создайте файл `.env` в корне проекта на основе примера:

```bash
# === LLM Настройки ===
LLM_PROVIDER=openrouter
MODEL_NAME=deepseek/deepseek-chat
OPENROUTER_API_KEY=sk-or-v1-ваш_ключ

# === Эмбеддинги (локальная модель) ===
EMBEDDING_MODEL=BAAI/bge-small-en-v1.5

# === Пути ===
CHROMA_DB_PATH=./chroma_db
DATA_DIR=/app/data
PYTHONUNBUFFERED=1
```


### Структура

```
data/
├── benefits.txt          # Преимущества стажировки
├── deadlines.txt         # Сроки подачи заявок
├── germany_rules.txt     # Правила для Германии
├── france_rules.txt      # Правила для Франции
└── general_info.txt      # Общая информация
```

### Запуск

```
docker-compose down -v
docker-compose up --build
```
